Crafty Bay is a modern e-commerce mobile application built with Flutter, offering a seamless shopping experience for users. The app is fully powered by **REST API integration**, enabling real-time product browsing, cart management, user authentication, and order processing. It provides an intuitive and responsive interface for both Android and iOS devices.

## Sign In Screen
This screen allows existing users to log in. It displays a welcoming **"Welcome Back"** message with email and password input fields. A teal **Login** button provides a clear call to action, and a **Sign Up** link navigates new users to the registration screen. The design is clean, modern, and matches the app branding.

---

## Sign Up Screen
New users can register on this screen by providing their email, first and last name, mobile number, address, and password. A clear **Sign Up** button completes registration, while a **Back to Login** link allows navigation to the login page. The interface is simple, user-friendly, and intuitive.

---

## Home Page (Category Slider)
The home page is the main landing screen for browsing products and promotions. It includes the app logo, profile, and notification icons in the header, along with a prominent search bar. A banner carousel highlights featured items, and a category grid allows quick filtering. Products are displayed with images, names, prices, ratings, and a wishlist icon. The bottom navigation bar provides quick access to Home, Categories, Cart, and Wishlist. Thematic sections such as New, Special, and Popular are included with **See all** links.

## Cart Screen
Users can review and manage selected products. Each item displays its image, name, details (size/color), and price. Quantity controls (+/-) and a delete option are provided. The total price is displayed at the bottom with a **Checkout** button for proceeding to payment.

---

## Payment Gateway Screen
This screen allows users to finalize purchases through multiple payment options. Organized tabs provide selection for **Cards**, **Mobile Banking**, and **Net Banking**. Local popular options like bKash and Rocket are prominently displayed. A large **PAY** button confirms the transaction.
